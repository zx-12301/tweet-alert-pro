## 📋 需要你配合的配置清单

### 🔴 必须配置（核心功能）

#### 1. Twitter API 凭证 ⭐⭐⭐
**用途**: 监控 Twitter 用户推文  
**配置位置**: http://localhost:3000/settings → Twitter API

**配置步骤**:
1. 访问 https://developer.twitter.com/en/portal/dashboard
2. 申请开发者账号（选择 Hobby 免费计划）
3. 创建项目和应用
4. 获取 4 个凭证：
   - API Key
   - API Key Secret  
   - Access Token
   - Access Token Secret
5. 填入配置页面并保存

**预计时间**: 15-30 分钟

---

#### 2. Twilio 电话通知 ⭐⭐⭐
**用途**: 拨打真实语音电话通知  
**配置位置**: http://localhost:3000/settings → 语音电话

**配置步骤**:
1. 访问 https://www.twilio.com/try-twilio
2. 注册账号（免费$15 额度）
3. 验证邮箱和手机号
4. 获取凭证：
   - Account SID
   - Auth Token
5. 获取试用电话号码
6. 在 Twilio 控制台验证你的手机号（重要！）
7. 填入配置页面并保存

**预计时间**: 10 分钟

**注意**: 试用账号只能拨打已验证的号码

---

#### 3. SMTP 邮件服务 ⭐⭐⭐
**用途**: 发送真实邮件通知  
**配置位置**: http://localhost:3000/settings → 邮件服务

**推荐方案 A: Gmail**
1. 登录 Gmail
2. 访问 https://myaccount.google.com/apppasswords
3. 生成应用专用密码
4. 配置参数：
   ```
   SMTP 服务器：smtp.gmail.com
   端口：587
   用户名：your-email@gmail.com
   密码：应用专用密码
   发件人：your-email@gmail.com
   ```

**推荐方案 B: 163 邮箱**
1. 登录 163 邮箱
2. 设置 → POP3/SMTP/IMAP
3. 开启 SMTP 服务
4. 获取授权码
5. 配置参数：
   ```
   SMTP 服务器：smtp.163.com
   端口：587
   用户名：your-email@163.com
   密码：授权码
   发件人：your-email@163.com
   ```

**预计时间**: 5-10 分钟
